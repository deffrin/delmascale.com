WEIGH BRIDGE

Electronic Truck Weighing Bridge

  • Model/Type : DTS 200
  • Version : Surface Mounted / Pit Mounted
  • Platform Type : Fully steel fabricated / Concrete.
  • Capacities : 10 Tonnes to 200 Tonnes
  • Platform size : 6x3m,12x3m,18x3m,21x3m
Special capacities and Platforms sizes can be ordered.

  • Fabricated steel platform module (6m*3m) each
  • Compression type Load cells (30/40/50 Ton)
  • Self aligning rocket support assemblies
  • Coping angles
  • Junction box

Optional
  • Application Software Package for weighing, ticketing and reporting.

Weigh Bridge instrumentation

All the DTS-200 weigh bridges can be connected to a wide range of instrumentations from a simple weigh bridge indicator or terminal to a sophisticated PC-based data management system.
Weigh Bridge indicator having different specifications can be ordered The weigh bridge indicator which can be interfaced with a PC or a Printer.
PC based unique Delma Weigh Application software package for weighing, ticketing and reporting.
Also, the instrumentation can be interfaced directly to a Printer to get the weigh ticket with the information.

Weigh Bridge platform

Mild Steel welded construction based on different capacities to form platform of different sizes.

Weigh Bridge Load Cells

The DTS-200 truck weigh bridge is supplied with Compression type or Double ended Shear Beam Type Load cells

Weigh Bridge Networking Software

The software can be networked, user defined and password protected for supervisor applications.

The basics of the Software:

  • Name and Address of the company.
  • Serial Number, In and Out – Date and Time.
  • Vehicle Number – Source and Destination.
  • Client, Product.
  • Gross, Tare and Net Weight.
  • Password protected to login/ user control/ Manual weighing/ Edit ticket/ Gross mode.
  • Edited Ticket can be identified.
  • Extra four user-defined fields can be used for additional information as required by you.
  • Data can be generated both in Text and Excel format which can be used for MIS.
  • Excel Reports can be extracted for ticket report, customer report and material report.
  • Additional ticket generations are possible (as duplicate).
